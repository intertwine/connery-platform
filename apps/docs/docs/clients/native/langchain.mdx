# LangChain

Use the Connery plugin infrastructure in your applications built with LangChain.
Run the Connery actions from agents, chatbots, or other applications.

## Tools and Toolkits

Connery is natively integrated into LangChain through the Tools and Toolkits.

Check out the LangChain documentation below for more information and examples on
integrating Connery actions in your application with LangChain.
