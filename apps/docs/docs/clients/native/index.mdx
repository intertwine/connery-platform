# Native clients

Below is a list of native clients build and maintained by Connery.
Clients allow you to use actions from your runner across different platforms.
There are two types of native clients: Application Clients and API Clients.

:::note Helpful resources

- [What is a client?](/docs/runner/introduction/core-concepts#client)

:::

## Application Clients

Application Clients are user-facing applications in various platforms that allow running actions from your runner.

- [OpenAI GPTs](/docs/clients/native/openai/gpt)
- [Slack App](/docs/clients/native/slack)
- [Make App](/docs/clients/native/make)

## API Clients

API Clients enable the development of new Application Clients or use the Connery plugin infrastructure in other applications.

- [OpenAI Assistants API](/docs/clients/native/openai/assistant)
- [LangChain](/docs/clients/native/langchain)
- [API](/docs/clients/native/api)
- [CLI](/docs/clients/native/api)
